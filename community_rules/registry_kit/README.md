# guardmarly-community-rules Registry Kit (Track 4)

This folder is a ready-to-publish starter kit for the dedicated `guardmarly-community-rules` repository.

## Includes

- Rule schema and contribution guidance
- Starter packs (auth, SSRF/open redirect, secrets/config)
- CI workflow for rule-pack validation

## Recommended publish steps

1. Create new GitHub repository: `guardmarly-community-rules`
2. Copy this kit into repo root
3. Enable Actions and run first validation workflow
4. Configure issue templates and sponsorship/bounty links
