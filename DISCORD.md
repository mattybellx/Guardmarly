# Discord Community (Coming Soon)

> A place for Ansede users and contributors to connect, ask questions, and share findings.

## Planned Channels

| Channel | Purpose |
|---------|---------|
| `#announcements` | Release notes, blog posts, events |
| `#general` | Introductions and general discussion |
| `#help` | Ask questions about using Ansede |
| `#rules` | Discuss and share community detection rules |
| `#contributing` | Development discussions, PR reviews |
| `#showcase` | Share what you found with Ansede |
| `#security-research` | Discuss vulnerabilities and responsible disclosure |
